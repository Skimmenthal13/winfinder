Privacy Policy

Last updated: June 4, 2026  ·  Effective immediately

Short version Win Finder does not collect, store, or transmit any personal data. The app runs entirely on your Mac, with no server-side component of any kind.

1. Who we are

Win Finder is an open source macOS application developed and maintained by Vincenzo Schimmenti (@Skimmenthal13). The source code is publicly available at github.com/Skimmenthal13/winfinder under the MIT License.

2. Data we do NOT collect

Win Finder does not collect any data. Specifically, the application does not collect, transmit, or store:

  • Your name, email address, or any other personal identifier
  • File names, folder structures, or any content of your file system
  • Usage statistics, crash reports, or telemetry of any kind
  • IP addresses or device identifiers
  • Location data

3. How the app works

Win Finder operates entirely on your local machine. It reads your file system to display folders and files, but this data never leaves your device. There are no analytics frameworks, no crash reporting services, and no network connections initiated by the application itself.

4. macOS permissions

Win Finder may request access to your file system in order to function as a file manager. This permission is handled entirely by macOS and the data remains on your device at all times. You can revoke this permission at any time via System Settings → Privacy & Security.

5. Third-party services

The Win Finder website is hosted on GitHub Pages. When you visit this website, GitHub may collect standard web server logs (IP address, browser type, referring URL) in accordance with GitHub's Privacy Statement. Win Finder has no control over and no access to this data.

6. Downloads

DMG downloads are served directly from GitHub Releases. The same GitHub privacy terms apply. No download tracking is performed by Win Finder.

7. Extension system

Win Finder supports a local extension system via JSON files placed in ~/.config/winfinder/actions/. These files are read locally by the app and are never transmitted anywhere. Extensions are entirely under your control.

8. Children's privacy

Win Finder does not collect any data from anyone, including children under 13. There is nothing to collect.

9. Changes to this policy

If this policy ever changes in a meaningful way, the updated date at the top of this page will reflect the change and a note will be added to the GitHub repository.

10. Contact

Questions about this privacy policy? Open an issue on GitHub or contact via the repository.
