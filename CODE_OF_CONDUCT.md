# Code of Conduct

## Our Pledge

Win Finder is an open source project built for people who switched from Windows to Mac and want a more familiar file management experience. We welcome contributors from all backgrounds and experience levels.

We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

**Positive behavior includes:**
- Being respectful and constructive in code reviews and issue discussions
- Welcoming newcomers and helping them get started
- Focusing on what is best for the project and its users
- Gracefully accepting feedback and criticism
- Acknowledging different perspectives and use cases

**Unacceptable behavior includes:**
- Insulting, demeaning, or harassing comments
- Personal attacks or political arguments unrelated to the project
- Trolling or deliberate provocation
- Publishing others' private information without consent
- Any conduct that would be inappropriate in a professional setting

## Scope

This Code of Conduct applies in all project spaces: GitHub Issues, Pull Requests, Discussions, and any other channel where project members interact.

## Enforcement

Instances of unacceptable behavior may be reported by opening a private issue or contacting the maintainer directly via GitHub. All reports will be reviewed and handled with discretion.

Maintainers have the right to remove, edit, or reject comments, commits, issues, and other contributions that do not align with this Code of Conduct, and to temporarily or permanently ban contributors for behaviors they deem inappropriate.
